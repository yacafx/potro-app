## How to use it

1. Run

```
      npm install
```

2. Create database and their user on https://mlab.com
3. Update cnx string at [config.js](config.js) file with the user, pass, dbname and domain from mlab.com
4. Run

```
      nodemon server.js
```

5. Test it on http://localhost:8080
